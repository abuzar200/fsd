Concept Summary

Bootstrap uses a 12-column grid. If you set a class to col-4, it takes up 4/12 (one-third) of the available width. Responsive prefixes like -md- (medium/tablets) and -lg- (large/desktops) specify at which screen size these rules should apply. This allows the layout to seamlessly adapt to any device without writing complex custom CSS media queries!
